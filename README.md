# nitro-fan
- A simple utility to manually control the fans of a Acer Nitro AN16-42 laptop
- Works probably only on Acer Nitro AN16-42 the EC probes were just modified from my original omen fan scripts.
- Also has a service that actively adjusts the fan speed according to tempertatures (cause the default BIOS control sucks)
- Made for a dear friend of mine to make his Linux gaming experience better.

# WARNING
- Use this branch at your own risk.

# Documentation
- Use `nitro-fan help` to see all available subcommands
